# DO_NOT_REPEAT

## 强制禁止重复的问题

- 不要让执行线程维护全局状态。
- 不要让执行线程写复杂管理文档。
- 不要让执行线程自己宣布 PASS。
- 不要没有读取相关记忆就做同类任务。
- 不要没有任务分级就开多线程。
- 不要让重任务按最低量级糊弄过去。
- 不要让轻任务被过度组织化。
- 不要 T4/T5 缺 Goal。
- 不要让子线程忘记 goal_id。
- 不要用空泛高级感替代具体视觉。
- 不要在创意任务里只给形容词。
- 不要在排版任务里平均铺满元素。
- 不要没有视觉锚点就直接生成图像。
- 不要没有 Skill Scout 就猜本机 Skill。
- 不要让幕僚长做审核。
